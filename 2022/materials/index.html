CHIRIMEN ハンズオン教材

CHIRIMEN とは、ウェブブラウザーや Node.js の JavaScript からハードウェア制御が可能な IoT システム開発のプロトタイピング環境です。デジタルのソフトとフィジカルなハードを連携するデバイスを JavaScript だけで容易に実現できます。

Raspberry Pi 4 (Model B) を使った CHIRIMEN 環境での実習

Raspberry Pi 上のブラウザーでコードの編集と実行ができるので、Raspberry Pi 一式があれば、PC を用意することなく実習が可能です。別途、Raspberry Pi に接続する HDMI モニター、USB キーボード、USB マウスが必要です。

Raspberry Pi Zero を使った CHIRIMEN 環境での実習

Raspberry Pi Zero 上の Node.js を JavaScript 処理系として動作します。コード編集や環境設定は PC 上のブラウザーから USB 経由で行え、Raspberry Pi Zero と PC があれば別途開発環境は必要ありません。
