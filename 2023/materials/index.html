Raspberry Pi Zero W 版 CHIRIMEN ハンズオン

誰でもできる IoT 開発基礎 CHIRIMEN Raspberry Pi Zero W チュートリアル (お試し編)

Raspberry Pi Zero 上の Node.js を JavaScript 処理系として動作する Raspberry Pi Zero 版の CHIRIMEN を短時間で試していただくためのチュートリアル動画です。コード編集や環境設定は PC 上のブラウザーから USB 経由で行え、Raspberry Pi Zero と PC があれば別途開発環境は必要ありません。

実習資料

解説内容は チュートリアルサイト の内容に沿って実施していきます。

Raspberry Pi Zero W 版 CHIRIMEN ハンズオン 導入～応用編

誰でもできる IoT 開発基礎 CHIRIMEN Raspberry Pi Zero W チュートリアル (ハンズオン講習会編)

2021 年 11 月に徳島県で開催された「Web×IoT メイカーズチャレンジ PLUS in 徳島」の IoT ハンズオン講習会 (遠隔講習) のアーカイブ動画です。

{% include youtube-video.inc.html id="Y0OaDyrnHV0" %}

実習資料

以下、CHIRIMEN チュートリアルサイト の基礎編から実施していきます。

  • 基礎編: ① L チカをしてみよう、② センサーを使ってみよう
  • 遠隔制御編: ③ 遠隔制御してみよう、④ 自動起動に対応してみよう
  • 応用編: ⑤ 色々なセンサーを使ってみよう、⑥ 色々なセンサーをリモート対応にしてみよう

Raspberry Pi 4 版 CHIRIMEN ハンズオン

誰でもできる IoT 開発基礎 Raspberry Pi 4 版 CHIRIMEN ハンズオン (講習会アーカイブ)

Raspberry Pi 上のブラウザーでコードの編集と実行を行う Raspberry Pi 4 (Model B) 版の CHIRIMEN 環境でのハンズオン講習の動画です。 Raspberry Pi に HDMI モニター、USB キーボード、USB マウスを直接接続して実習を行うため、ノート PC を使わずに講習を行うことができます。

2021 年 12 月に鳥取県で開催された「Web×IoT メイカーズチャレンジ PLUS in 鳥取」の IoT ハンズオン講習会 (遠隔講習) のアーカイブ動画です。

{% include youtube-video.inc.html id="dnwsNB8PXKA" %}

実習資料

以下、CHIRIMEN チュートリアルサイト の基礎編から実施していきます。

  • 基礎編: ① L チカしてみよう (初めての GPIO)、② GPIO の使い方、③ センサーを使ってみよう (初めての I2C)
  • 応用編: ④ 色々なセンサーを使ってみよう
