Web×IoT メイカーズチャレンジ PLUS 教材集

Web×IoT メイカーズチャレンジ PLUS は、標準技術やオープンソースソフトウェア (OSS) 利活用についての理解を深めながら、シングルボードコンピュータ等を使ったハンズオン形式の講習会やハッカソンでの開発体験を通し、IoT システム開発の実践的なスキルアップの機会提供を行う取り組みです。

Web×IoT メイカーズチャレンジ PLUS 中央実行委員会では、皆様の学習や教育活動にお役立ていただけるよう、再利用可能なライセンス (クリエイティブ・コモンズ・ライセンス) のもとで講習資料や講義動画の一部を公開しております。

講義資料の改変や再配布も可能ですので、出典元を明記の上でご活用ください。

なお、以下に掲載する教材以外についても、こちらの GitHub リポジトリ 上で公開しております。

ハンズオン講習 誰でもできる IoT 開発基礎
～ CHIRIMEN Raspberry Pi Zero W チュートリアル ～

CHIRIMEN とは、Web ブラウザや Node.js の JavaScript からハードウェア制御が可能な IoT システム開発のプロトタイピング環境です。デジタルのソフトとフィジカルなハードを連携するデバイスを JavaScript だけで容易に実現できます。

この動画では、2021 年 11 月に徳島県で開催された「Web×IoT メイカーズチャレンジ PLUS in 徳島」で宮崎 典行氏 (KDDI 株式会社 / Web×IoTメイカーズチャレンジ PLUS 中央実行委員) が講師を担当した IoT ハンズオン講習会のアーカイブ動画です。

教材として Raspberry Pi Zero W を使用し、Web 標準技術で の IoT 開発の基礎知識を実践的に学ぶ内容です。

基礎編

  • L チカしてみよう
  • センサーを使ってみよう

遠隔制御編

  • 遠隔制御してみよう
  • 自動起動に対応してみよう

応用編

  • 色々なセンサーを使ってみよう
  • 色々なセンサーをリモート対応にしてみよう
