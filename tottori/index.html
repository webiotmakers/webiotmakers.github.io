Web × IoT メイカーズチャレンジ 2017 in 鳥取

概要

小型ボードコンピューター Raspberry Pi 3 を使って、物理デバイス (モノ) を JavaScript からコントロールする「WebGPIO」や「I2C」といった低レベル API の操作などを実践的に学ぶハンズオン講習会を行います。講習会の 3 週間後には、学んだ知識を活用し 2 日間で実際に作品を作ってその出来を競い合う「ハッカソン = Hackathon」イベントを開催します。

ハッカソンでの作品制作の材料費補助やチームに 1 台 Raspberry Pi 3 のプレゼントなどの特典もあります！

なお、ハッカソンでの作品制作に向けて、11/19 の講習会後半に鳥取大学 EDGE とのコラボ企画として『IoT を活用した鳥取地域課題解決アイデアワークショップ』も行います。こちらは、「プログラミングはできないけれどアイデア出しは得意！」「デザインはちょっとできる…」といった非エンジニアの方にも参加いただけます。

エンジニアにとっては、実践的なスキルを身につけるチャンス！ 非エンジニアの方にとっても自分のアイデアやデザインが、エンジニアの手でカタチになるかもしれない貴重な機会です。大学生や専門学校生はもちろん、若手社会人、高校生の方もぜひお気軽にご参加ください。
